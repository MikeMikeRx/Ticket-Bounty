# Author Notes

## Motivation

This project was built while studying *The Road to Next* by Robin Wieruch. The goal was to explore Next.js as a full-stack framework and understand the complete development workflow.

The project focuses on key Next.js concepts such as static vs. dynamic rendering, server vs. client components, server-side rendering, server actions, server queries, and cursor-based pagination. It also includes database integration using Prisma ORM.

## The Road to Next

This book builds on the previous *The Road to React*, which covers the fundamentals of React.js. My earlier project, [My-Hacker-Stories](https://github.com/MikeMikeRx/My-Hacker-Stories), was created while working through the first part of that learning path.

## The Goal

I initially started web development as a React developer but quickly wanted to understand how full-stack systems are built. Next.js provided a practical platform to explore modern web architecture and better understand common industry patterns and standards.
